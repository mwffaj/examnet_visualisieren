📊 Klassenauswertung

Schreibverläufe einer ganzen Klasse sammeln und gemeinsam visualisieren.

1
Pro Schüler:in — Antwortverlauf öffnen, dann klicken:
📥 Schüler:in hinzufügen ← in die Lesezeichenleiste ziehen
2
Nach allen Schüler:innen — Klasse anzeigen:
📊 Klassenauswertung ← in die Lesezeichenleiste ziehen
3
Daten zurücksetzen (auf exam.net ausführen):
🗑 Daten löschen ← in die Lesezeichenleiste ziehen
💾 Die Daten werden lokal im Browser gespeichert (localStorage) und bleiben bis zum manuellen Löschen erhalten. Nichts wird an externe Server gesendet.
